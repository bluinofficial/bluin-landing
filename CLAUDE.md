# BLU'IN 프로젝트 규칙

## 토큰 절약 규칙 (매 세션 필수)
- 세션 시작 시 항상 /compact 실행
- 작업 요청은 번호 목록으로 간결하게
- 30분 작업마다 /compact 실행
- 새 작업 시작 전 /clear 실행
- 파일 읽을 때 필요한 부분만 읽기
- 한 번에 최대 3가지 수정만 요청

## 프로젝트 정보
- 서비스명: BLU'IN (블루인)
- 서비스: 스쿠버다이빙 플랫폼
- 정식 런칭: 2026년
- 구글플레이: https://play.google.com/store/apps/details?id=com.app.bluin
- 앱스토어: 출시 예정 (비활성화 처리)
- 로고 경로: BLU'IN_Logo/PNG/

## 이미지 규칙
- 기본은 다이빙/해양 관련 이미지 우선 사용
- 퀄리티 좋은 이미지라면 다른 주제도 기용 가능
- 저작권 없는 이미지만 사용 (무조건)
  · 허용: Unsplash, Pexels, Pixabay, CC0 라이선스
  · 금지: 저작권 있는 유료 이미지, 출처 불명 이미지
- 이미지 변경 시 반드시 URL + 출처 목록을 먼저 나에게 보여줄 것
- 내가 육안으로 확인하고 승인한 이미지만 적용
- 승인 전에는 절대 index.html에 반영하지 말 것

## 코딩 규칙
- HTML/CSS/JS 단일 파일 유지
- 모바일 반응형 필수
- 컬러: 딥 네이비 + 청록 + 골드
